新人团活动规则
  • 团购活动为移动端会员专享，用户注册成为西客商城会员之后即可发起团购。
  • 同一用户可发起多个新人团活动邀请好友，参团用户必须是新注册的用户，单一活动商品限购一件。
  • 同一用户的判定原则，包括但不限于注册账户、身份证号、设备号、支付宝账号等。
  • 新人团组团成功后获得的该商品的优先购买权，由于商品库存有限，以先付款先得原则，所以请尽快支付。
  • 新人团活动参团后将不能退出组团，但是组团成功后不强制购买。付款后不支持修改订单地址。
  • 本活动不参与其他优惠活动，付款后不支持修改订单地址。
  • 新人团活动是基于好友的转发扩散，获得团购优惠，为了保证广大消费者的权益，西客商城有权判定为恶意倒货的团购解散并取消订单。如在活动期间发现违规行为（重复购买、无效手机号参与等），西客商城有权取消违规订单，封闭账号。
  • 在法律允许范围内，西客商城保留最终解释权。
